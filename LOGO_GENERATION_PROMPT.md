# Logo Generation Prompt für Memocracy

## Prompt für ChatGPT / DALL-E / Midjourney

```
Erstelle ein modernes, tech-orientiertes Logo für "Memocracy" - eine AI-powered Memecoin Community Platform.

**Brand Name:** Memocracy
**Tagline:** "AI-Powered Memecoin Intelligence"

**Design-Anforderungen:**

1. **Stil:**
   - Modern, jung, tech-orientiert
   - Clean und minimalistisch
   - Futuristisch aber zugänglich
   - Ähnlich zu modernen AI/Tech Startups (wie Anthropic, OpenAI, Notion)
   - Gradient-Farben (Blau, Lila, Cyan)
   - Glassmorphism oder Neumorphism Elemente

2. **Themen die integriert werden sollen:**
   - **Dezentralität:** Netzwerk-Knoten, Verbindungen, Blockchain-ähnliche Strukturen
   - **Voting/Polls:** Abstimmungs-Symbole, Checkmarks, Stimmzettel
   - **Founding Wallets:** Wallet-Symbole, Finanzierung, Crowdfunding
   - **Community:** Menschen, Gruppen, Verbindungen
   - **AI/Tech:** Neuronale Netzwerke, Datenpunkte, intelligente Systeme

3. **Visuelle Elemente (kombiniere mehrere):**
   - Gehirn/Neural Network + Abstimmungs-Symbol
   - Wallet + Netzwerk-Knoten
   - Abstimmungs-Hand + Blockchain-Verbindungen
   - Moderne geometrische Formen (Kreise, Hexagone, Knoten)
   - Gradient-Farbverläufe (Blau → Lila → Cyan)
   - Transparenz/Glass-Effekte

4. **Farbpalette:**
   - Primär: Blau (#3B82F6, #2563EB)
   - Sekundär: Lila (#8B5CF6, #7C3AED)
   - Akzent: Cyan (#06B6D4, #0891B2)
   - Hintergrund: Weiß oder dunkles Grau/Schwarz
   - Optional: Grüne Akzente für "Voting/Approval"

5. **Typografie (falls Text enthalten):**
   - Modern, Sans-Serif
   - Futuristisch aber lesbar
   - "M" könnte als Icon verwendet werden
   - Optional: "MEMOCRACY" als Wortmarke

6. **Format:**
   - Quadratisch (für Favicon/App Icon)
   - Funktioniert auch als horizontales Logo
   - SVG-optimiert (vektorisiert)
   - Funktioniert in verschiedenen Größen (16x16 bis 512x512)

7. **Inspiration:**
   - Brain/Neural Network (für AI)
   - Abstimmungs-Hand oder Checkmark (für Voting)
   - Wallet oder Coin-Symbol (für Finanzierung)
   - Netzwerk-Knoten (für Dezentralität)
   - Moderne Tech-Logos wie: Anthropic, OpenAI, Notion, Linear

8. **Konkrete Ideen:**
   - **Option A:** Gehirn-Icon mit Abstimmungs-Checkmarks als Neuronen
   - **Option B:** Wallet-Symbol mit Netzwerk-Verbindungen
   - **Option C:** Abstimmungs-Hand mit Blockchain-Knoten im Hintergrund
   - **Option D:** "M" als abstraktes Symbol mit Gradient und Netzwerk-Elementen
   - **Option E:** Kombination aus Gehirn + Wallet + Abstimmungs-Symbol

9. **Technische Details:**
   - Minimalistisch genug für kleine Größen
   - Erkennbar auch ohne Farben (funktioniert in Schwarz-Weiß)
   - Skalierbar ohne Qualitätsverlust
   - Modern und zeitlos (nicht zu trendig)

10. **Mood & Feeling:**
    - Vertrauenswürdig
    - Innovativ
    - Community-orientiert
    - Tech-forward
    - Zugänglich (nicht zu komplex)

**Output:** 
Erstelle mehrere Varianten (3-5) mit unterschiedlichen Ansätzen. Jede Variante sollte:
- Als vollständiges Logo (Icon + Text)
- Als reines Icon (für Favicon)
- In verschiedenen Farbvarianten (Light/Dark Mode)
- Mit und ohne Text

**Zusätzliche Anforderungen:**
- Das Logo sollte auf der Webseite funktionieren (Header, Favicon, Social Media)
- Es sollte zu einer modernen, gradient-basierten Webseite passen
- Es sollte Memecoin-Community ansprechen (jung, dynamisch)
- Es sollte auch professionell genug für Business-Kontexte sein
```

## Alternative: Detaillierter Prompt für spezifische Tools

### Für DALL-E / Midjourney:

```
Create a modern, tech-oriented logo for "Memocracy" - an AI-powered memecoin community platform. 

The logo should combine:
- A brain or neural network symbol (representing AI)
- Voting/checkmark elements (representing community voting)
- Wallet or coin symbols (representing founding wallets)
- Network nodes or connections (representing decentralization)

Style: Modern, minimalist, futuristic, similar to Anthropic or OpenAI logos. 
Color palette: Blue (#3B82F6) to Purple (#8B5CF6) to Cyan (#06B6D4) gradients.
Background: White or transparent.

The logo should be:
- Clean and scalable
- Work as both icon and wordmark
- Suitable for web and mobile apps
- Recognizable at small sizes (favicon)
- Modern and youthful but professional

Format: Square, vector-style, glassmorphism effects optional.
```

### Für ChatGPT (Text-to-Design Beschreibung):

```
Beschreibe ein Logo-Design für "Memocracy":

**Hauptelement:** Ein abstraktes "M" oder Gehirn-Symbol, kombiniert mit:
- Abstimmungs-Checkmarks als dekorative Elemente
- Netzwerk-Knoten/Verbindungen im Hintergrund
- Wallet-Symbol als subtiles Detail

**Design-Stil:**
- Minimalistisch und modern
- Gradient von Blau über Lila zu Cyan
- Glassmorphism-Effekte
- Geometrische, klare Linien
- Ähnlich zu modernen Tech-Startups

**Layout:**
- Zentriertes Icon
- Optional: "MEMOCRACY" Text darunter
- Funktioniert quadratisch (Icon) und horizontal (mit Text)

**Farben:**
- Primär: Blau (#3B82F6)
- Sekundär: Lila (#8B5CF6)  
- Akzent: Cyan (#06B6D4)
- Transparenz-Effekte

**Zielgruppe:** Junge Memecoin-Community, Tech-Enthusiasten, DeFi-User
**Gefühl:** Innovativ, vertrauenswürdig, community-orientiert, tech-forward
```

## Verwendung

1. **Für ChatGPT (DALL-E Integration):** Nutze den ersten Prompt
2. **Für Midjourney:** Nutze den "Detaillierter Prompt für spezifische Tools"
3. **Für Designer-Briefing:** Nutze alle drei Prompts als Inspiration

## Zusätzliche Tipps

- Erstelle mehrere Varianten und teste sie auf der Webseite
- Stelle sicher, dass das Logo auch in Schwarz-Weiß funktioniert
- Teste verschiedene Größen (Favicon, Header, Social Media)
- Berücksichtige Dark Mode Varianten
- Stelle sicher, dass es zu deinem aktuellen Design-System passt (Gradient-Farben, etc.)
